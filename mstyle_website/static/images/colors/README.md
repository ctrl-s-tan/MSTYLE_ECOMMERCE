# Color Images Directory

This directory should contain color variant images for products.

## File Naming Convention:
- Use lowercase color names
- Replace spaces with underscores
- Use .jpg or .png format

## Examples:
- red.jpg
- blue.jpg
- black.jpg
- white.jpg
- dark_blue.jpg
- light_green.jpg

## Image Requirements:
- Size: 60x60px (minimum)
- Format: JPG or PNG
- Show the actual product in that color variant
- Good lighting and clear visibility

## Fallback:
If a color image is not found, the system will fall back to a solid color circle.

## Profile Management

### Overview
Manage your rider profile, vehicle information, and account settings.

### Accessing Your Profile
1. Click on your profile icon in the top navigation bar
2. Select "Profile" from the dropdown menu
3. Your profile page will display

### Profile Information Sections

#### 1. Personal Information
**Editable Fields:**
- First Name
- Last Name
- Email Address (verified)
- Phone Number
- Profile Picture
- Emergency Contact Name
- Emergency Contact Number

**How to Update:**
1. Click the "Edit Profile" button
2. Modify the desired fields
3. Click "Save Changes"
4. Confirmation message appears

**Profile Picture Guidelines:**
- Clear, professional photo
- Face clearly visible
- Good lighting
- No filters or effects
- Recent photo (within 6 months)

#### 2. Vehicle Information
**Required Details:**
- **Vehicle Type**
  - Motorcycle
  - Bicycle
  - Car
  - Scooter

- **Vehicle Model**
  - Make and model
  - Example: Honda Wave 125

- **Plate Number**
  - Official registration number
  - Format: ABC-1234

- **Year Model**
  - Manufacturing year
  - Example: 2020

- **Vehicle Color**
  - Primary color
  - Helps customers identify you

**How to Update:**
1. Go to "Vehicle Information"
2. Click "Edit"
3. Update vehicle details
4. Upload new documents if changed
5. Save changes
6. Admin verification required

#### 3. Verification Documents
**Required Documents:**
1. **Valid Government ID**
   - Driver's License (preferred)
   - National ID
   - Passport
   - Status: Verified/Pending/Rejected

2. **Vehicle OR/CR**
   - Official Receipt
   - Certificate of Registration
   - Must be current and valid
   - Status indicator shown

3. **NBI Clearance**
   - National Bureau of Investigation clearance
   - Must be recent (within 6 months)
   - Status indicator shown

**Document Status:**
- ✓ Verified (Green) - Approved and active
- ⏳ Pending Review (Yellow) - Under admin review
- ✗ Rejected (Red) - Needs resubmission
- ⚠ Expiring Soon (Orange) - Renew within 30 days

**Resubmit Documents:**
1. Click on rejected/expired document
2. Upload new document
3. Ensure clear and readable
4. Submit for review
5. Wait for admin approval

#### 4. Bank Account Information
**For Earnings Payout:**
- Bank Name
- Account Number
- Account Name (must match your name)
- Branch

**How to Add/Update:**
1. Navigate to "Payment Settings"
2. Click "Add Bank Account" or "Edit"
3. Enter bank details accurately
4. Verify account information
5. Save changes

**Important:**
- Account name must match your registered name
- Double-check account number
- Keep information updated
- Required for receiving earnings

### Availability Settings

#### Set Your Working Hours
**Configure Availability:**
1. Go to "Availability Settings"
2. Toggle "Available for Deliveries"
3. Set working hours:
   - Start time
   - End time
   - Days of the week
4. Save schedule

**Availability Status:**
- **Online** (Green): Available for deliveries
- **Busy** (Yellow): Currently on delivery
- **Offline** (Gray): Not accepting deliveries

**Quick Toggle:**
- Use dashboard toggle to go online/offline
- Instant status change
- Visible to system for assignment

### Security Settings

#### Change Password
1. Click "Security Settings"
2. Enter current password
3. Enter new password
4. Confirm new password
5. Click "Update Password"

**Password Requirements:**
- Minimum 8 characters
- At least one uppercase letter
- At least one number
- At least one special character

#### Two-Factor Authentication (2FA)
1. Enable 2FA in security settings
2. Scan QR code with authenticator app
3. Enter verification code
4. Save backup codes securely

### Notification Preferences
**Configure Notifications:**
- New delivery assignments
- Pickup reminders
- Delivery updates
- Payment received
- Customer messages
- Seller messages
- System announcements

**Notification Channels:**
- Email notifications
- SMS alerts
- Push notifications
- In-app notifications

### Best Practices
- Keep contact information current
- Update vehicle information if changed
- Renew documents before expiration
- Use professional profile picture
- Enable 2FA for security
- Set realistic availability hours
- Keep emergency contact updated

- **Purple**: At Seller Location
- **Blue**: Picked Up / In Transit
- **Cyan**: Arrived at Customer
- **Green**: Delivered

### Delivery Workflow

#### Step 1: Heading to Seller
**Actions:**
1. Accept delivery
2. Review pickup location
3. Click "Navigate"
4. Follow GPS directions
5. Update status when arrived

**Tips:**
- Check traffic conditions
- Call seller if needed
- Verify pickup address
- Plan parking spot
- Estimate arrival time

#### Step 2: At Seller Location
**Pickup Process:**
1. Arrive at seller location
2. Update status to "Arrived at Seller"
3. Contact seller if needed
4. Verify package details
5. Check items against order
6. Inspect package condition
7. Take photo of package (optional)
8. Update status to "Picked Up"

**Verification Checklist:**
- Correct order ID
- Package sealed properly
- Fragile items marked
- Quantity matches
- Special instructions noted
- Invoice included

**If Issues Arise:**
- Package not ready
- Items missing
- Wrong items
- Damaged packaging
- Contact seller immediately
- Report issue in app
- Wait for resolution
- Document with photos

#### Step 3: In Transit
**During Delivery:**
1. Update status to "In Transit"
2. Navigate to customer location
3. Handle package carefully
4. Follow traffic rules
5. Update ETA if delayed
6. Notify customer if needed

**Safe Delivery Practices:**
- Secure package properly
- Protect from weather
- Avoid rough handling
- Keep fragile items stable
- Maintain package integrity

**Communication:**
- Update customer on progress
- Notify if running late
- Confirm delivery details
- Ask for access instructions
- Professional communication

#### Step 4: Arrived at Customer
**Arrival Process:**
1. Arrive at delivery location
2. Update status to "Arrived"
3. Find parking safely
4. Locate exact address
5. Contact customer if needed
6. Prepare package for handover

**If Customer Not Available:**
- Call customer
- Wait reasonable time (5-10 minutes)
- Send message
- Contact support if no response
- Follow failed delivery protocol

#### Step 5: Delivery Completion
**Handover Process:**
1. Greet customer professionally
2. Verify customer identity
3. Confirm order details
4. Hand over package
5. Request signature/photo
6. Collect payment (if COD)
7. Update status to "Delivered"
8. Thank customer

**Proof of Delivery:**
- Customer signature
- Photo of delivered package
- Photo with customer (optional)
- Delivery location photo
- Timestamp automatically recorded

**COD Payment Collection:**
1. Inform customer of amount
2. Collect exact payment
3. Provide change if needed
4. Count money carefully
5. Confirm amount in app
6. Issue receipt
7. Secure payment safely
